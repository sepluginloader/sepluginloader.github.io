Plugin Hub

The Plugin Hub is where you can find all the content that developers have put on Plugin Loader. Developers can either publish plugins with third-party libraries and unrestricted API access or client mods from the workshop. Client mods are regular mods from the workshop that Plugin Loader can load regardless of whether the mod is on the server or not.
