--- title: Installing Plugin Loader ---

Space Engineers Launcher

To install Plugin Loader, a custom game launcher must be installed. The game launcher will automatically download and start Space Engineers with Plugin Loader.

  1. Download the SpaceEngineersLauncher.exe file from the link above
  2. Make sure the exe file is not blocked before continuing by opening the file properties of the extracted file and checking the Unblock box (if it exists)
  3. The exe file can now be placed in the game Bin64 folder
    • You can find the Bin64 folder by right clicking on Space Engineers in Steam and selecting Properties. Then under the Local Files tab, select Browse and navigate to the Bin64 folder
  4. Launch Space Engineers by opening the SpaceEngineersLauncher.exe file

Troubleshooting

I have graphics issues after installing Plugin Loader

Make sure your display is set to use the correct GPU in the game display options. Also make sure SpaceEngineersLauncher.exe is set to high performance mode in your GPU's control panel.

For other issues, create an issue on the github page or join the discord server
